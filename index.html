CSS stands for Cascading Style Sheets. CSS is the language we use to style an HTML document. CSS describes how HTML elements should be displayed. CSS saves a lot of work. It can control the layout of multiple web pages all at once!

HTML stands for Hyper Text Mark-up Language. HTML is the language we use to produce a document. HTML describes how it's elements should be used. HTML is used to make many webpages, if not, all of the 3trillion pages it holds together: the World Wide Web!

Javascript is represented by p5.js*. Javascript is the language we use to create games and more activity-riddled pages and apps. Javascript describes how games are made. Javascript is used to make many games, whether on the Web, your phone or in your life!
